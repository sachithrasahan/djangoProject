======================Requirements to execute the project======================
1. python
2. django framework
3. mysql database 
4. mysql client for python

======================Commands to execute the project==========================
1. start the database - name(profilemanager)
2. open project folder
3. add database credintials to settings.py
4. open vscode terminal under the project folder and execute below
5. run "python manage.py makemigrations"
6. run "python manage.py migrate"
7. run "python manage.py runserver"
